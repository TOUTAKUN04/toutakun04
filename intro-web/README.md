# Intro Web Portfolio

A responsive personal portfolio website for **Touta Kun** built with plain HTML, CSS, and JavaScript.

## Highlights

- Animated hero section with particle background
- Typing-text intro animation
- Glassmorphism-inspired UI sections
- Auto theme support (follows system light/dark) with manual toggle button
- Mobile-friendly navigation and layout
- Animated project cards with GitHub and release/readme links
- Contact call-to-action section

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Font Awesome
- Google Fonts (Inter, Fira Code)

## Project Structure

```text
intro-web/
  index.html
  style.css
  script.js
  images/
    project1.jpg
    project2.jpg
    project3.gif
```

## Run Locally

No build tool is required.

1. Open the `intro-web` folder.
2. Open `index.html` in your browser.

Or use VS Code Live Server for auto-reload while editing.

## Customization

- Update text/content in `index.html`
- Update styles/animations in `style.css`
- Update behavior/interactions in `script.js`
- Replace project assets in `images/`
